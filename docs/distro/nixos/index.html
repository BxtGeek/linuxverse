About NixOS

NixOS is a unique Linux distribution built on the Nix package manager, offering declarative, reproducible system configuration. Every aspect of the system is described in configuration files.

Recommended For

  • DevOps engineers
  • Reproducible builds
  • Configuration management
  • Advanced users
