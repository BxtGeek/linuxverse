About Whonix

Whonix is a desktop operating system designed for advanced security and privacy. It uses a two-VM model — a Gateway routing traffic through Tor and a Workstation for user activity — to prevent IP leaks.

Recommended For

  • Maximum anonymity
  • Security research
  • Running alongside Tor
